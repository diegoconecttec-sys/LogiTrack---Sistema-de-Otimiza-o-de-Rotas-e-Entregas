# 🚚 Sistema Logístico com IA

Sistema inteligente para otimização de rotas de entrega em tempo real utilizando Inteligência Artificial.

## 📌 Funcionalidades

* Otimização de rotas
* Rastreamento em tempo real
* Previsão de entrega (ETA)
* Monitoramento de frota

## 🏗 Arquitetura

* Microsserviços
* Event-Driven (Kafka)
* PostgreSQL + PostGIS
